# payroll-system-excel
A simple and automated Excel-based Payroll Calculator for managing hourly wages, attendance, overtime, and monthly salary.


This repository contains a detailed and automated **payroll-system-excel** built in Microsoft Excel.  
It helps calculate **hourly wages, weekly hours, overtime, bonuses, and total monthly salary** for all employees.

The sheet also includes **summary statistics** such as maximum, minimum, average, and total values for wages, hours, and monthly payroll.
